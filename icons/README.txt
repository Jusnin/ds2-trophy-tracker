放置你的獎盃圖示於此資料夾。
Put your trophy icons in this folder.

命名規則 / Naming:
    trophy-01.png   →  No.01  黑暗靈魂 (Platinum)
    trophy-02.png   →  No.02  喪失的記憶
    trophy-03.png   →  No.03  國王戒指
    ...
    trophy-38.png   →  No.38  黑暗靈魂之始

- 編號需補零至兩位數 (01, 02 … 38)。
- 建議使用正方形圖片 (例如 256x256)，png / jpg 皆可 (副檔名請用 .png)。
- 沒有放圖的獎盃會自動顯示一個營火占位符號，放入後重新整理即可顯示。
